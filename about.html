About This Project

A proof-of-concept for free web hosting with GitHub Pages

What is GitHub Pages?

GitHub Pages is a free static site hosting service provided by GitHub. It takes HTML, CSS, and JavaScript files directly from a repository and publishes them as a website.

Every GitHub user gets a free domain at username.github.io. You can also host project-specific sites at username.github.io/repo-name. Custom domains are supported too.

Key Features

Free HTTPS: Every GitHub Pages site comes with a free SSL certificate, so your site is served securely over HTTPS by default.

Custom Domains: You can point your own domain name to your GitHub Pages site with a simple DNS configuration.

Jekyll Support: GitHub Pages natively supports Jekyll, a static site generator, so you can use templates, layouts, and Markdown content.

Getting Started

To deploy this site on GitHub Pages:

1. Create a new repository on GitHub (e.g., sample_web).

2. Push these files to the repository's main branch.

3. Go to Settings > Pages, select "Deploy from a branch", and choose main.

4. Your site will be live at https://fz-tm.github.io/sample_web/ within a minute.
